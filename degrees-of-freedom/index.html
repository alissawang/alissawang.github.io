Knowing the row totals, every cell except the last one in each row is free to hold any value.

By the same token, every cell except the last in each column is free to vary given the column total.

The degrees of freedom is therefore the number of rows - 1 multiplied by the number of columns - 1. In this case, 2 x 2 = 4 degrees of freedom..
